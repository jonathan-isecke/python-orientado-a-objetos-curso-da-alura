# python-orientado-a-objetos-curso-da-alura

## Curso: Aprenda a programar em Python com Orientação a Objetos

Este repositório reúne todos os códigos, projetos e exercícios realizados no curso de programação da Alura.
Ao longo da formação, aprofundei conhecimentos essenciais para desenvolvimento em Python, incluindo:

* Programação orientada a objetos (POO)
* Extração e manipulação de dados via URL
* Uso de coleções avançadas do Python
* Validação de dados seguindo padrões internacionais
* Prática de testes unitários e boas práticas de qualidade de código

Ao todo, foram 505 atividades práticas, consolidando o aprendizado com exemplos reais e cenários de uso comuns no mercado.
